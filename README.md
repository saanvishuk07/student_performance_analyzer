# student_performance_analyzer

# 📊 STUDENT PERFORMANCE ANALYSIS SYSTEM

---

##  OVERVIEW
This project is a Python-based data analysis system designed to evaluate student performance using marks and attendance stored in a CSV file. It computes key statistical measures such as mean, average, minimum, maximum, and standard deviation, assigns grades, and visualizes the results using graphs.

The system helps educators and analysts quickly understand performance trends and student distribution.

---

##  FEATURES
-  Load student data from CSV file  
-  Calculate:
-  Mean / Average
-  Minimum & Maximum  
-  Standard Deviation  
-  Automatic grade assignment  
-  Graphical visualization:
- Bar graph for marks
- Scatter graph for attendance  
-  Simple and beginner-friendly  
-  Quick performance insights  

---

##  TOOLS USED
- **Python** – Programming language  
- **Pandas** – Data processing and analysis  
- **Matplotlib** – Data visualization  
- **CSV File** – Data storage format
